# 补光灯 Web 应用

这是一个本地可运行的补光灯 Web 应用，使用原生 HTML、CSS 和 JavaScript 编写，采用了现代化的设计风格。

## 功能特性

- **单页应用**：页面打开即可使用
- **中央显示区域**：显示当前选中的颜色
- **全屏模式**：点击大色块或全屏按钮切换全屏模式
- **色块选择**：
  - 预设颜色（不可删除，可拖拽调整顺序）
  - 自定义颜色（可添加、删除）
- **颜色选择器**：支持直观的颜色选择
- **状态记忆**：使用 localStorage 保存选中颜色、颜色顺序和自定义颜色
- **手势操作**：支持滑动手势切换颜色

## 文件结构

```
code/
├── index.html      # HTML 结构
├── style.css       # 样式文件
└── script.js       # JavaScript 逻辑
```

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用任何静态服务器运行项目

## 交互说明

### 颜色选择
- 点击任意色块切换中央显示颜色
- 当前选中的色块会显示为"选中状态"（略微缩小并有边框）

### 全屏模式
- 点击中央色块或全屏按钮进入/退出全屏模式
- 全屏模式下仅显示颜色，隐藏色块选择栏

### 自定义颜色
- 点击 "+ " 按钮显示颜色选择器
- 在颜色选择器中选择颜色后，点击"添加"按钮将颜色添加到自定义色块列表
- 长按自定义色块显示删除按钮，点击可移除

### 拖拽排序
- 长按系统预设颜色可拖拽调整顺序

### 滑动手势
- 在颜色面板上左右滑动可快速切换颜色

### 颜色选择确认
- 颜色选择器中新增"添加"按钮，点击后才将颜色添加到列表
- 点击"添加"按钮后，颜色选择器会自动收起
- 点击颜色选择器外部区域也可收起窗口

## 技术细节

- 不使用任何前端框架或第三方库
- 所有代码均为原生 JavaScript
- 使用 localStorage 实现状态持久化
- 响应式设计，适配移动端
- 现代化的 UI 设计风格
- 优化的触摸操作体验
- 支持长按删除自定义颜色
